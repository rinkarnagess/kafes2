Дипломный проект интернет-магазина кофе и аксессуаров: главная страница, каталог товаров и страница товара.
Реализовано на Vue 3 + Vite (без бекенда).

Функционал (MVP)

Реализовано:

Главная страница (hero, преимущества, блоки)

Каталог товаров (рендер из данных)

Фильтр по категории (кофе / аксессуары) + поиск по названию

Страница товара (/product/:id)

Адаптивная шапка с бургер-меню (Vue)

⏳ В планах (следующие контрольные точки):

Корзина (localStorage)

Изменение количества/удаление товаров

Оформление заказа (форма + валидация)

Адаптивность и улучшение UI


Технологии:

Vue 3/,
Vite,
Vue Router,
HTML / CSS,
JavaScript,
Git

Установка и запуск
1) Установить зависимости
   npm install

2) Запуск в режиме разработки
   npm run dev

3) Сборка проекта
   npm run build

4) Предпросмотр сборки
   npm run preview
